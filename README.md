Projeto final do curso de Java do Projeto Mais Pra Ti / Alfamidia.
Programa de cadastro de Aluno e professores, para inclusão de notas para Alunos, cadastro somente em memoria sem deixar gravado em banco de dados.
Aluno pode acessar sistema, fazer seu cadastro, alterar algumas Informações do cadastro e consultar suas notas.
Professor pode acessar sistema, fazer seu cadastro, alterar algumas Informações do cadastro, atribuir e consultar as notas dos alunos.
Diretor pode acessar sistema, cadastrar e consultar Alunos e Professores, conultar e alterar as notas dos alunos.
